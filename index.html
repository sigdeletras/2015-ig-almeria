Taller "Fuentes de Información Geográfica para la provincia de Almería"

Almería, de mayo de 2015 :: HackLab Almería y Geoinquietos Almería

Obtetivo del taller

Hacer un repaso a las principales páginas web oficiales de descarga de información geográfica para la provincia de Almería.

Nos centramos en la IG geográfica no en los datos que podemos vincular a la información geográfica.

Buscaremos información para poder presentar especialmente una variable.

Datos

  • Datos con referencias geográficas: Domicilio, referencia catastral, distrito, barrio, municipio, mancomunidad...Mejor ejemplo una hoja de cálculo con coordenadas.
  • Datos ćon representación geográfica. Los datos con representación geográfica necesitando programas específicos como CAD o SIG.

En este taller nos centraremos en encontrar datos para no tener que generarlos.

Información en Internet (portales)

Desde hace unos años las administraciones públicas (directivas, leyes y normativas) publican sus repositorio y servicios de información geográfica en Internet.

  • Datos.
  • Servicios. Pueden usarse como base cartográfica. Ej. WMS (Fotografía áreas, Series cartográficas) o teselas/tiles (Google Map, OpenStreetMap...)

Formatos

  • Shapefile (SHP). Formato multiarchivo propietario (ESRI). Formato SIG por excelencia.
  • GeoJSON. GeoJSON is an open standard format for encoding collections of simple geographical features along with their non-spatial attributes using JavaScript Object Notation.

  • KML.

    <?xml version="1.0" encoding="UTF-8"?>
<kml xmlns="http://www.opengis.net/kml/2.2">
<Document>
<Placemark>
  <name>New York City</name>
  <description>New York City</description>
  <Point>
    <coordinates>-74.006393,40.714172,0</coordinates>
  </Point>
</Placemark>
</Document>
</kml>

  • GPX

  • CSV
  • ...

Grandes posibilidades en el campo del turismo, rutas, localización de servicios municipales...

Posibilidad de usar API para construir mapas interactivos en Internet por ejemplo MAPEA, CDAU...
